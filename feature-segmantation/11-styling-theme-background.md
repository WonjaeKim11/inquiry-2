# 스타일링 / 테마 / 배경 — 요구사항 명세서

> **문서번호**: FSD-011 | **FR 범위**: FR-015 ~ FR-017
> **라이선스**: Community (모든 스타일링 기능)

---

## 1. 목적/배경

Formbricks는 설문 UI의 시각적 커스터마이징을 위해 계층적 스타일링 시스템을 제공한다. Project(프로젝트) 레벨에서 전체 테마를 설정하고, 개별 Survey(설문) 레벨에서 해당 테마를 오버라이드할 수 있다. 이 시스템을 통해 브랜드 일관성을 유지하면서도 특정 설문에 대해 독립적인 디자인 적용이 가능하다.

스타일링 속성은 색상(Color), 타이포그래피(Typography), 레이아웃(Layout), 배경(Background), 카드(Card), 프로그레스바(Progress Bar), 로고(Logo) 등 40개 이상의 세분화된 항목으로 구성된다.

---

## 2. 범위 (In-scope / Out-of-scope)

### In-scope
- Project 레벨 기본 테마 설정 (프로젝트 스타일링 구조)
- Survey 레벨 스타일 오버라이드 (설문 스타일링 구조)
- 배경 4가지 타입: color, image, upload, animation
- Dark Mode 토글
- 카드 배치(Card Arrangement) 설정
- 로고 설정 및 숨김
- 프로그레스바 커스터마이징
- Brand Color 기반 자동 색상 생성(Suggest Colors)
- Legacy 스타일 필드에서 신규 필드로의 자동 마이그레이션

### Out-of-scope
- CSS 직접 편집
- 커스텀 폰트 업로드 (폰트 패밀리는 문자열 지정만 가능)
- Survey 런타임에서의 동적 스타일 변경

#### 4.3 카드 배치 옵션 (Card Arrangement)

카드 배치는 세 가지 스타일 옵션을 제공한다:

- casual: 카드가 약간 기울어진 스타일
- straight: 카드가 정렬된 스타일
- simple: 단순 스타일
- Link Survey와 App Survey에 대해 독립적으로 설정 가능

### FR-016: 배경 설정 (Background)

**4가지 배경 타입**:

| 타입 | bg 값 | 설명 |
|------|-------|------|
| color | CSS 색상 코드 (예: #ffffff) | 단색 배경 |
| image | Unsplash 이미지 URL | Unsplash 라이브러리에서 선택 |
| upload | 업로드된 이미지 URL (필수) | 직접 이미지 업로드 |
| animation | 애니메이션 식별자 | 동적 배경 애니메이션 |

- brightness: 배경 밝기 조절 (기본값 100)
- upload 타입의 경우 bg 값이 필수 (유효성 검증)
- Unsplash 연동 여부, 스토리지 설정 여부 플래그로 각 배경 타입 활성화 여부 결정

### FR-017: Project vs Survey 레벨 스타일링

#### Project 레벨 테마 (프로젝트 스타일링)

- 스타일 오버라이드 허용 여부: 개별 설문에서 스타일 오버라이드를 허용할지 결정
- 프로젝트의 Look & Feel 설정 페이지에서 관리

#### Survey 레벨 오버라이드 (설문 스타일링)

- 테마 스타일 오버라이드 여부: true이면 프로젝트 테마 대신 설문 자체 스타일 사용
- false이면 프로젝트 테마가 적용됨
- 설문 에디터의 Styling 탭에서 토글로 제어

#### 스타일 적용 우선순위

시스템 기본값 → Project 스타일링 → Survey 스타일링 (테마 오버라이드 활성화 시)

적용 순서:
1. 시스템 기본값
2. 프로젝트 레거시 필드 마이그레이션
3. 프로젝트 설정
4. 설문 레거시 필드 마이그레이션
5. 설문 설정 (최우선)

#### 로고 설정

- 로고는 URL과 배경색으로 구성
- Link Survey에서만 로고 및 배경 설정 가능
- 로고 숨김 여부로 로고 숨김 제어

### Suggest Colors (자동 색상 생성)

자동 색상 생성 기능은 단일 Brand Color에서 전체 색상 팔레트를 자동 생성한다:

- **질문 텍스트 색상**: Brand Color + #000000 (35% 혼합) -- 어두운 텍스트 색상
- **인풋 배경**: Brand Color + #ffffff (92% 혼합) -- 밝은 인풋 배경
- **인풋 테두리**: Brand Color + #ffffff (60% 혼합) -- 중간 테두리
- **카드 배경**: Brand Color + #ffffff (97% 혼합) -- 거의 흰색 카드 배경
- **페이지 배경**: Brand Color + #ffffff (85.5% 혼합) -- 페이지 배경
- **버튼 텍스트**: Brand Color 밝기에 따라 어두운(#0f172a) 또는 밝은(#ffffff) 자동 결정
